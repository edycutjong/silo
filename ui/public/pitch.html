Slide 04 // Core Architecture

Terminal 3 ADK Integration

Silo integrates 5 key Terminal 3 Host APIs directly. Evidence is ingested using `stash`, phone numbers are verified via `otp`, follow-up chat is blinded via `http-with-placeholders`, and report manifests are attested via `signing`.

stash Safe blind document storage
otp Credentials verified, never stored
http-placeholders Blinds relayed chat variables
signing Issues verifiable credentials
